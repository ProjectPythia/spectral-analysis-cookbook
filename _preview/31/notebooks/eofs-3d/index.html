Overview¶

Empirical Orthogonal Function (EOF) analysis identifies dominant patterns of variability in space-time data. Starting from a synthetic 2D field that evolves in time, we reshape the data into a matrix, compute covariance, extract eigenmodes, and interpret the results as either principal components (temporal analysis) or oscillation modes (spatial analysis).

  1. Mathematical background of EOF / PCA on space-time data

  2. Generate a synthetic surface and a time-varying amplitude

  3. Build and analyze the processing matrix (temporal covariance)

  4. Repeat the analysis using spatial covariance

  5. Compare EOF maps and principal component time series

Mathematical background¶

Space-time data matrix¶

Suppose we observe a field Z(x,y,t)Z(x, y, t)Z(x,y,t) on a regular grid with nx×nyn_x \times n_ynx​×ny​ spatial points and ntn_tnt​ time steps. After flattening each snapshot, we arrange the data in a processing matrix X∈Rm×nt\mathbf{X} \in \mathbb{R}^{m \times n_t}X∈Rm×nt​, where m=nxnym = n_x n_ym=nx​ny​ and each column is one time snapshot:

X=[∣∣∣x1x2⋯xnt∣∣∣]\mathbf{X} = \begin{bmatrix} | & | & & | \\ \mathbf{x}_1 & \mathbf{x}_2 & \cdots & \mathbf{x}_{n_t} \\ | & | & & | \end{bmatrix}X=⎣⎡​∣x1​∣​∣x2​∣​⋯​∣xnt​​∣​⎦⎤​
(1)

EOF analysis is equivalent to principal component analysis (PCA) applied to the columns or rows of X\mathbf{X}X, depending on which covariance matrix we choose.

Covariance matrices¶

Two covariance matrices can be formed from the same data matrix:

Ctime=XTX(nt×nt)\mathbf{C}_{\text{time}} = \mathbf{X}^{\mathsf T}\mathbf{X} \qquad (n_t \times n_t)Ctime​=XTX(nt​×nt​)
(2)
Cspace=XXT(m×m)\mathbf{C}_{\text{space}} = \mathbf{X}\mathbf{X}^{\mathsf T} \qquad (m \times m)Cspace​=XXT(m×m)
(3)

  • Ctime\mathbf{C}_{\text{time}}Ctime​ captures temporal co-variability (each entry compares two time indices across all spatial points).

  • Cspace\mathbf{C}_{\text{space}}Cspace​ captures spatial co-variability (each entry compares two spatial locations across all times).

In practice we diagonalize the smaller matrix for computational efficiency.

Eigenvalue problem¶

Let C\mathbf{C}C denote either covariance matrix. EOF analysis solves the standard eigenvalue problem:

C vk=λk vk\mathbf{C}\,\mathbf{v}_k = \lambda_k\,\mathbf{v}_kCvk​=λk​vk​
(4)

where λk\lambda_kλk​ is the variance explained by mode kkk and vk\mathbf{v}_kvk​ is the corresponding eigenvector.

The fraction of explained variance for mode kkk is

pk=λk∑jλjp_k = \frac{\lambda_k}{\sum_j \lambda_j}pk​=∑j​λj​λk​​
(5)

The interpretation of vk\mathbf{v}_kvk​ depends on which covariance matrix was used (see the two analysis paths below).

Projections: EOFs and principal components¶

Temporal analysis (C=XTX\mathbf{C} = \mathbf{X}^{\mathsf T}\mathbf{X}C=XTX):

  • Eigenvectors vk∈Rnt\mathbf{v}_k \in \mathbb{R}^{n_t}vk​∈Rnt​ are principal component time series.

  • Spatial EOF patterns are obtained by projecting the data onto each eigenvector:

ψk=X vk∈Rm\boldsymbol{\psi}_k = \mathbf{X}\,\mathbf{v}_k \in \mathbb{R}^{m}ψk​=Xvk​∈Rm
(6)

Reshaping ψk\boldsymbol{\psi}_kψk​ back to (nx,ny)(n_x, n_y)(nx​,ny​) gives an EOF map.

Spatial analysis (C=XXT\mathbf{C} = \mathbf{X}\mathbf{X}^{\mathsf T}C=XXT):

  • Eigenvectors vk∈Rm\mathbf{v}_k \in \mathbb{R}^{m}vk​∈Rm are oscillation modes (spatial patterns).

  • Reshape vk\mathbf{v}_kvk​ directly to obtain EOF maps.

  • Associated principal component time series are:

ak(t)=XT vk∈Rnta_k(t) = \mathbf{X}^{\mathsf T}\,\mathbf{v}_k \in \mathbb{R}^{n_t}ak​(t)=XTvk​∈Rnt​
(7)

The data can be approximated as a sum of outer products of spatial and temporal modes.

Generate synthetic data¶

Define the spatial surface¶

We start with a smooth 2D surface that depends only on (x,y)(x, y)(x,y). This is our basic spatial structure. We choosed to use the following function:

Z=cos⁡(x)cos⁡(y)e−x2+y2/4Z = \cos(x) \cos(y) e^{-\sqrt{x^2 + y^2}/4}Z=cos(x)cos(y)e−x2+y2​/4
(8)
# Function to generate a 2D surface
def surface(x, y):
    return np.cos(x) * np.cos(y) * np.exp(-np.sqrt(x**2 + y**2) / 4)

Build the processing matrix¶

We now construct the space-time data and reshape it into the matrix X\mathbf{X}X introduced above.

For each time index ttt:

Zt(x,y)=Z(x,y) cos⁡(t)and then shifted along xZ_t(x, y) = Z(x, y)\,\cos(t) \quad \text{and then shifted along } xZt​(x,y)=Z(x,y)cos(t)and then shifted along x
(9)

The 3D-data has shape (nx,ny,nt)(n_x, n_y, n_t)(nx​,ny​,nt​). Flattening the first two dimensions gives X\mathbf{X}X with shape (m,nt)(m, n_t)(m,nt​).

data_cube = np.zeros([Z.shape[0], Z.shape[1], len(time_multiplier)])
data_cube.shape

# Loop over the time indices to create the 3D-data
for time_index, multiplier in enumerate(time_multiplier):

    plt.grid(True)
    ax.set_xlabel("Time index")

Summary¶

We applied EOF analysis to a synthetic space-time field. The key steps were:

  1. Arrange snapshots in a processing matrix X\mathbf{X}X.

  2. Form either Ctime=XTX\mathbf{C}_{\text{time}} = \mathbf{X}^{\mathsf T}\mathbf{X}Ctime​=XTX or Cspace=XXT\mathbf{C}_{\text{space}} = \mathbf{X}\mathbf{X}^{\mathsf T}Cspace​=XXT.

  3. Solve C vk=λk vk\mathbf{C}\,\mathbf{v}_k = \lambda_k\,\mathbf{v}_kCvk​=λk​vk​ and rank modes by explained variance pkp_kpk​.

  4. Interpret eigenvectors as temporal PCs or spatial modes, then project to obtain the complementary EOF maps or PC time series.

The choice of covariance matrix determines which patterns are extracted directly as eigenvectors and which must be computed by projection.
