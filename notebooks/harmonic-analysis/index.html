Overview¶

The seasonal cycle is the most prominent mode of variability in many climate variables. In this notebook we use harmonic analysis, representing a time series as a sum of sine and cosine waves, to describe the monthly climatology of sea surface temperature (SST) at a single location, closely following Section 10.4 of Wilks (2019)

We will:

  1. Load NOAA Optimum Interpolation SST V2 daily data and build a monthly climatology.

  2. Approximate the climatology with a single cosine wave using an “eyeballed” amplitude and phase.

  3. Estimate the amplitude and phase of the first harmonic by least squares using the analysis equations (Wilks (2019) Eqs. 10.60 and 10.59).

  4. Extend the analysis to higher harmonics and quantify how much of the variance each harmonic explains.

  5. Reconstruct the climatology exactly using all n/2 harmonics.

  6. Use the phase-shift trick of Wilks Ex. 10.10 to evaluate the fitted cosine on a daily time axis.

A single cosine wave¶

A simple harmonic representation of a series of length nnn is:

yt  =  yˉ + C1cos⁡ ⁣(2πtn−ϕ1),t=1,2,…,n,y_t \;=\; \bar{y} \,+\, C_1 \cos\!\left(\frac{2\pi t}{n} - \phi_1\right), \qquad t = 1, 2, \dots, n,yt​=yˉ​+C1​cos(n2πt​−ϕ1​),t=1,2,…,n,
(1)

with three parameters: the mean yˉ\bar{y}yˉ​, the amplitude C1C_1C1​, and the phase ϕ1\phi_1ϕ1​. The cosine is preferred over the sine because ϕ1\phi_1ϕ1​ then has the clean interpretation “angle of the maximum”, so the function peaks at t=ϕ1 n/2πt = \phi_1\, n / 2\pit=ϕ1​n/2π.

As a first guess we set:

  • yˉ\bar{y}yˉ​ = sample mean of the climatology,

  • C1C_1C1​ = half the peak-to-peak range, and

  • ϕ1\phi_1ϕ1​ corresponds to the month containing the warmest mean SST.

y = sst_monthly_climatology.values
t = sst_monthly_climatology["month"].values.astype(float)

As we see, the eyeballed cosine fit is not a good fit for the data. We will now use least squares to fit the first harmonic to the data.

Least-squares first harmonic¶

Using the trigonometric identity:

cos⁡(α−ϕ)=cos⁡ϕcos⁡α+sin⁡ϕsin⁡α\cos(\alpha-\phi) = \cos\phi\cos\alpha + \sin\phi\sin\alphacos(α−ϕ)=cosϕcosα+sinϕsinα
(2)

We can rewrite the equation of the previous section as a linear combination of an unshifted cosine and an unshifted sine:

yt  =  yˉ + A1cos⁡ ⁣(2πtn) + B1sin⁡ ⁣(2πtn),A1=C1cos⁡ϕ1,B1=C1sin⁡ϕ1.y_t \;=\; \bar{y} \,+\, A_1 \cos\!\left(\tfrac{2\pi t}{n}\right) \,+\, B_1 \sin\!\left(\tfrac{2\pi t}{n}\right), \qquad \begin{aligned} A_1 &= C_1 \cos\phi_1, \\ B_1 &= C_1 \sin\phi_1. \end{aligned}yt​=yˉ​+A1​cos(n2πt​)+B1​sin(n2πt​),A1​B1​​=C1​cosϕ1​,=C1​sinϕ1​.​
(3)

In this form A1A_1A1​ and B1B_1B1​ are ordinary regression slopes, they could be obtained by feeding cos⁡(2πt/n)\cos(2\pi t/n)cos(2πt/n) and sin⁡(2πt/n)\sin(2\pi t/n)sin(2πt/n) as predictors into any linear regression model.

For data that are evenly spaced with no missing values (our case), the least-squares estimates collapse to the “analysis equations”:

A1=2n∑t=1nytcos⁡ ⁣(2πtn),B1=2n∑t=1nytsin⁡ ⁣(2πtn),A_1 = \frac{2}{n}\sum_{t=1}^{n} y_t \cos\!\left(\tfrac{2\pi t}{n}\right), \qquad B_1 = \frac{2}{n}\sum_{t=1}^{n} y_t \sin\!\left(\tfrac{2\pi t}{n}\right),A1​=n2​t=1∑n​yt​cos(n2πt​),B1​=n2​t=1∑n​yt​sin(n2πt​),
(4)

and the amplitude/phase:

C1=A12+B12,ϕ1=atan2⁡(B1, A1).C_1 = \sqrt{A_1^2 + B_1^2}, \qquad \phi_1 = \operatorname{atan2}(B_1,\, A_1).C1​=A12​+B12​​,ϕ1​=atan2(B1​,A1​).
(5)
omega_t = 2.0 * np.pi * t / n

A1 = (2.0 / n) * np.sum(y * np.cos(omega_t))
B1 = (2.0 / n) * np.sum(y * np.sin(omega_t))

C1 = np.hypot(A1, B1)
phi1 = np.arctan2(B1, A1) % (2 * np.pi)

Higher harmonics¶

Just as adding more predictors to a multiple regression always improves the fit, adding more cosine waves of higher frequency always improves the fit of a harmonic representation. Any series of length nnn can be reproduced exactly by the following equation:

yt  =  yˉ + ∑k=1n/2[ Akcos⁡ ⁣(2πktn)+Bksin⁡ ⁣(2πktn)].y_t \;=\; \bar{y} \,+\, \sum_{k=1}^{n/2} \left[\, A_k \cos\!\left(\tfrac{2\pi k t}{n}\right) + B_k \sin\!\left(\tfrac{2\pi k t}{n}\right) \right].yt​=yˉ​+k=1∑n/2​[Ak​cos(n2πkt​)+Bk​sin(n2πkt​)].
(6)

The kkk-th harmonic completes exactly kkk full cycles over the nnn samples; its angular frequency is ωk=2πk/n\omega_k = 2\pi k/nωk​=2πk/n. For evenly spaced data the coefficients are given by the generalized analysis equations:

Ak=2n∑t=1nytcos⁡ ⁣(2πktn),Bk=2n∑t=1nytsin⁡ ⁣(2πktn),A_k = \frac{2}{n}\sum_{t=1}^{n} y_t \cos\!\left(\tfrac{2\pi k t}{n}\right), \qquad B_k = \frac{2}{n}\sum_{t=1}^{n} y_t \sin\!\left(\tfrac{2\pi k t}{n}\right),Ak​=n2​t=1∑n​yt​cos(n2πkt​),Bk​=n2​t=1∑n​yt​sin(n2πkt​),
(7)

with amplitude Ck=Ak2+Bk2C_k = \sqrt{A_k^2 + B_k^2}Ck​=Ak2​+Bk2​​ and phase ϕk=atan2⁡(Bk,Ak)\phi_k = \operatorname{atan2}(B_k, A_k)ϕk​=atan2(Bk​,Ak​).

For our monthly climatology n=12n=12n=12, so we can fit up to n/2=6n/2 = 6n/2=6 harmonics. Because nnn is even, the phase of the highest harmonic (k=n/2k=n/2k=n/2) is forced to zero.

Below we build a small helper that returns (Ak,Bk,Ck,ϕk)(A_k, B_k, C_k, \phi_k)(Ak​,Bk​,Ck​,ϕk​) for any kkk, then plot the first three harmonics separately.

def harmonic_coeffs(y, k):
    """
    Return (A_k, B_k, C_k, phi_k) for harmonic k of an evenly-spaced series.

    Implements Wilks Eqs. 10.64 (A_k, B_k) and 10.65 (C_k, phi_k).
    """
    n = len(y)
    t = np.arange(1, n + 1, dtype=float)
    omega_kt = 2.0 * np.pi * k * t / n
    A = (2.0 / n) * np.sum(y * np.cos(omega_kt))
    B = (2.0 / n) * np.sum(y * np.sin(omega_kt))
    C = np.hypot(A, B)
    phi = np.arctan2(B, A) % (2 * np.pi)
    return A, B, C, phi


coeffs = {k: harmonic_coeffs(y, k) for k in range(1, n // 2 + 1)}

def reconstruct(y_mean, coeffs, t_eval, n, k_max):
    """Sum of the first k_max harmonics evaluated at times t_eval."""
    out = np.full_like(np.asarray(t_eval, dtype=float), y_mean)
    for k in range(1, k_max + 1):
        A, B, *_ = coeffs[k]
        out += A * np.cos(2 * np.pi * k * t_eval / n) \
             + B * np.sin(2 * np.pi * k * t_eval / n)
    return out

Variance explained by each harmonic¶

For an evenly-spaced series the variance contributed by harmonic kkk is Ck2/2C_k^2 / 2Ck2​/2, and the total sample variance can be decomposed as

Var⁡(y)  =  ∑k=1n/2Ck 22.\operatorname{Var}(y) \;=\; \sum_{k=1}^{n/2} \frac{C_k^{\,2}}{2}.Var(y)=k=1∑n/2​2Ck2​​.
(8)

This is Parseval’s identity for a finite cosine series, and it is the discrete analogue of the power spectrum introduced in the ---- notebook. Plotting Ck2/(2Var⁡(y))C_k^2/(2 \operatorname{Var}(y))Ck2​/(2Var(y)) tells us how much of the seasonal cycle’s variability is captured by each harmonic individually.

ks       = np.array(list(coeffs.keys()))

Exact reconstruction with all n/2n/2n/2 harmonics¶

With n=12n = 12n=12 samples the sum of n/2=6n/2 = 6n/2=6 harmonics has 2×6=122 \times 6 = 122×6=12 free parameters — exactly enough to pass through every data point. This is an overfit in the previous regression analogy (one degree of freedom per data point), but here that is precisely the goal: we are representing the climatology, not forecasting future values.

full_recon = reconstruct(y_mean, coeffs, t, n, n // 2)

fig, ax = plt.subplots(figsize=(8, 4))
ax.plot(t, y, "ko", markersize=9, label="Data")

Summary¶

Using single Sea Surface Temperature grid point, we were able to reconstruct the monthly seasonal cycle of the time series using Harmonic Analysis. Hee we showed that:

  • A single cosine wave with eyeballed amplitude and phase already captures the main features of the seasonal cycle.

  • Replacing the eyeballed values with least-squares estimates via the analysis equations tightens the fit, especially for the phase.

  • Higher harmonics extend the representation to any shape; the cumulative reconstruction with 1, 2, 3, … harmonics converges toward the data.

  • Summing all n/2n/2n/2 harmonics reproduces the climatology exactly.

  • The fraction of variance carried by each harmonic, Ck 2/(2σy2)C_k^{\,2}/(2\sigma_y^2)Ck2​/(2σy2​), is a discrete line spectrum, the conceptual seed of the power spectrum studied in the --- notebook.

What’s next?¶

Now that we can represent a 12-month seasonal cycle as a sum of cosines, the next notebook, Remove the seasonal cycle using harmonic regression, generalizes this idea to daily time series: a small number of annual harmonics is fit by least squares and subtracted to obtain anomalies suitable for studying variability on other time scales.
